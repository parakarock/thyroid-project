อาหารที่สามารถรับประทานได้
  • กลุ่มข้าวแป้ง ท่านควรเลือกข้าวหรือแป้งที่ไม่ขัดขาว ได้แก่ ข้าวกล้อง ข้าวซ้อมมือ ข้าวไรซ์เบอรี่ ขนมปังโฮลวีท เป็นต้น
  • กลุ่มโปรตีน ท่านควรเลือกกินหมู ไก่ ไข่ขาว เนื้อปลาน้ำจืด ( ยกเว้นปลาร้า ) และธัญพืช เช่น ถั่ว ( ยกเว้น ถั่วเหลือง ถั่วแดง และถั่วขาว ) และข้าวโพด เป็นต้น
  • ผัก ท่านสามารถกินผักสดได้เกือบทุกชนิด ยกเว้น ถั่วฝักยาว หน่อไม้ฝรั่ง สาหร่าย ผักกาด หน่อไม้ กระถิน หัวไชเถ้า ฟักแฟง และกะหล่ำปลีดิบ
  • ผลไม้ ท่านสามารถกินผลไม้สดที่ไม่หวานจนเกินไป เช่น กล้วย ฝรั่ง แอปเปิ้ล ผลไม้อบแห้ง เช่นลูกเกด หรือดื่มน้ำผักหรือน้ำผลไม้ทดแทนได้
  • เครื่องปรุง ท่านสามารถใช้พริกไทย เครื่องเทศ หรือสมุนไพรแห้งหรือสดทุกชนิด ( ยกเว้น พริกชนิดเผ็ด ) เกลือหรือซ๊อสปรุงรสที่ไม่เติมไอโอดีนและใช้น้ำมันพืชได้ทุกชนิด ( รวมทั้งน้ำมันถั่วเหลือง ) ในการประกอบอาหาร
  • เครื่องดื่ม ท่านควรเลือกเครื่องดื่มประเภทชาสมุนไพร ( เช่น ตะไคร้ ดอกคำฝอย ขิง ) ทดแทนการดื่มชา กาแฟ เครื่องดื่มชูกำลัง และแอลกอฮอล์

  • ข้าวกล้องผัดผักตามชอบ อาจใส่ไข่ขาวแต่ไม่ใส่ไข่แดง
  • ปลาทอดหรือปลานึ่งที่อบหรือคั่วกับเกลือสินเธาว์ที่มีปริมาณไอโอดีนต่ำ
  • แกงจืดใช้น้ำต้มกระดูกหมูแทนซุปก้อน ปรุงรสด้วยพริกไทย น้ำตาล และเครื่องปรุงรสสำหรับผู้ที่ต้องจำกัดการบริโภคไอโอดีน
  • ต้มยำน้ำใส ใส่เห็ด มะเขือเทศ หอมใหญ่ ใส่เนื้อปลา ปรุงรสด้วยมะนาว พริก
  • สลัดผักหรือผลไม้ ราดน้ำสลัดแบบน้ำใส ( น้ำส้มสายชูผสมกับน้ำมัน
